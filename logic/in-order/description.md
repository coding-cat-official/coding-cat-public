Write a function `in_order(a: int, b: int, c: int) -> bool` that takes three integer inputs, `a`, `b`, and `c`. The function should return `True` if the integers are in strictly increasing order, otherwise return `False`.

For example:
- `in_order(1, 2, 3)` → `True`
- `in_order(3, 2, 1)` → `False`
- `in_order(2, 2, 3)` → `True`
