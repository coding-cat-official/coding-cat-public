Write a function `contains_pi(lst)` that takes a list of integers `lst` and returns `True` if the sequence `[3, 1, 4]` appears anywhere in the list. Return `False` otherwise.

For example:
- `contains_pi([1, 2, 3, 1, 4, 5])` should return `True` (the sequence [3, 1, 4] is present).
- `contains_pi([3, 1, 2, 4, 5])` should return `False` (the sequence [3, 1, 4] is not present).
- `contains_pi([3, 1, 4, 3, 1, 4])` should return `True` (the sequence [3, 1, 4] is present).
