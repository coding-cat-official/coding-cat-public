Write a function `triangle_area(base: float, height: float) -> float` that takes the `base` and `height` of a triangle as floats and calculates its area.

The area of a triangle can be calculated using the formula: `Area = 1/2 x base x height`

For example:
- `triangle_area(10.0, 5.0) → 25.0` as `1/2 x 10 x 5`.
- `triangle_area(7.0, 3.0) → 10.5` as `1/2 x 7 x 3`.
