Write a function `is_multiple(a: int, b: int) -> bool` that takes two integers, `a` and `b`. The function should return `True` if `a` is a multiple of `b`, and `False` otherwise. 

For example: 
- `is_multiple(10, 2) → True` because `10` is divisible by `2`


**Hint:** Solve [Is Even](https://coding-cat.club/#/problems/is_even) first, this is about having no remainder after a division.
