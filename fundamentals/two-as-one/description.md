Write a function `two_as_one(a: int, b: int, c: int) -> bool` that takes three integer inputs, `a`, `b`, and `c`. The function should return `True` if it is possible to add any two of the integers to get the third integer. If none of these conditions are met, return `False`.

For example:
- `two_as_one(2, 3, 5) → True` because `2 + 3 == 5`
- `two_as_one(1, 2, 4) → False` because no two of the integers add up to the third
