Write a function `number_is_palindrome(n: int) -> bool` that takes a two-digit integer and returns `True` if it is a palindromic number — that is, the two digits are the same.

For example:
- `number_is_palindrome(11) → True`
- `number_is_palindrome(56) → False`

**Hint**: You may want to convert the number to a string or use arithmetic to extract the tens and ones digits.
