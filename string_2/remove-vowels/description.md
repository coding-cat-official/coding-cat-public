In a given string, write a function `remove_vowels(strng: str) -> str` that returns said string but without any vowels. The string will always have at least one element and the inputted string will be lowercase. 

For example:
- `remove_vowels(“pranjwal”) → prnjwl`
- `remove_vowels(“veronika”) → vrnk` 
- `remove_vowels(“blind”) → blnd`
