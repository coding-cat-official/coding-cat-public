Write a function `double_odd(i_list)` that returns a list **ONLY** with the **ODD integers** values from the `i_list` that have an **ODD index**. The length of `i_list` is bigger or equal to 0.

For example:
- `double_odd([1, 2, 3]) → []`
- `double_odd([1, 1, 1]) → [1]`
- `double_odd([2, 1, 3, 7]) → [1, 7]`
