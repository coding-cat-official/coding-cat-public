Write a function `function2(num1, num2):` that takes in 2 numbers, `num1` and `num2` and returns their sum.

Parameters are where functions get a little more dynamic. Values are passed to the function when it is called, and then they are available in the function as variables.

Example:

`function2(10, 25)`
* This is calling function2 with two numbers, 10 and 25
* In this execution, `num1` = 10 and `num2` = 25

`print` statements can help you figure out what's going on inside functions when they run. Just be sure to remove them later once you know everything works!

Try writing `print(num1)` in function2 and see what happens!
