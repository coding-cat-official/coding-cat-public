Write a function `calculate_absolute(n: int) -> int` that takes an integer `n` as input and returns its absolute value. The absolute value of a number is its non-negative value.

For example:
- `calculate_absolute(-5)` → `5`
- `calculate_absolute(7)` → `7`
- `calculate_absolute(0)` → `0`
