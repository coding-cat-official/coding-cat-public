Write a function `is_even(num)` that takes an integer `num` as input and returns `True` if the integer is even, and `False` otherwise. 

For example:
- `is_even(4)` -> `True`
- `is_even(7)` -> `False`
- `is_even(0)` -> `True`

**Hint:** This problem is all about finding the remainder of a division. Recall the earlier "Math Operations 2" problem.
