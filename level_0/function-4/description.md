Python can be used like a calculator, if you know how to use math operators.

Most math operators are very straightforward:
- `+` for addition
- `-` for subtraction
- `*` for multiplication
- `/` for division (note: always returns a `float` to allow decimals)
- `**` for exponentiation

Some examples:
```
example_add = 20 + 32
example_sub = 33 - 9
example_mult = 10 * 7
example_div = 200 / 5
example_expo = 2 ** 8
example_pemdas = (5 + 1) * (11 - 4) / 3
```
